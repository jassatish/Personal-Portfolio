## Introduction

This is the source code for my personal portfolio website. The portfolio showcases my projects, skills, experience, and contact information. You can visit the live website [here](https://jassatish.github.io/Personal-Portfolio/).

## Features

- Home: Introduction and brief overview.
- About: Information about me, my background, and my interests.
- Projects: Display of my projects with descriptions and links to the source code.
- Skills: List of my technical skills.
- Experience: My work and educational experience.
- Contact: Form to get in touch with me.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap

## Installation

1. Clone the repository:
    ```sh
    git clone https://github.com/jassatish/Personal-Portfolio.git
    cd Personal-Portfolio
    ```

2. Open `index.html` in your web browser to view the portfolio locally.

## Usage

- Navigate through the different sections to learn more about me and my work.
- Check out my projects and visit the links to see the source code.
- Use the contact form to get in touch with me.
